Projects

Here is a collection of projects I have worked on.

Wrap It All

Spotify Wrapped-style desktop app that analyzes your email and calendar data to generate personalized year-in-review analytics with AI-powered insights.

  • Built with Electron, React, and Prisma/SQLite; integrates Gmail, Outlook, and Google Calendar via OAuth
  • Claude API for sentiment analysis, tone detection, and natural language chat over wrap data

DNS Tunnel

Tunnels HTTP traffic through DNS queries to access websites when normal internet traffic is blocked, using a Stop-and-Wait protocol for reliable data transfer.

  • Built client-server system in Python with Scapy for DNS packet crafting and Base64-encoded TXT record payloads
  • Implemented alternating-bit reliability protocol with Internet Checksum validation

Federated Learning Speech Reconstruction

Co-authored paper on reconstructing speech from federated learning gradients (ACCEPTED, Complex Networks 2024).

  • Implemented gradient-matching attack in PyTorch on LibriSpeech corpus to reconstruct spectrograms
  • Experimented with DeepSpeech1, various loss functions, and regularization techniques

Biographical RAG System

RAG system for conversations about historical figures using their actual writings and biographical content.

  • Content scraping pipeline for Wikisource, Project Gutenberg, speeches, and letters
  • Semantic search with ChromaDB; GPT-4 Q&A with source citations
